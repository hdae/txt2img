# 設定メモ

- `type`: モデルの利用モード
  - `sdxl`: SDXL
- `model`: モデルの指定
  - `urn:air:...`: CivitaiのAIR
  - `org/repo@...`: HuggingFaceのリポジトリ
  - `https://...`: URL
  - `file://...`: ローカルのパス
- `vae`: VAEの指定
  - `urn:air:...`: CivitaiのAIR
  - `org/repo@...`: HuggingFaceのリポジトリ
  - `https://...`: URL
  - `file://...`: ローカルのパス
  - `null`: 指定しない
- `loras`: LoRAの追加
  - 配列として記載する
    - `urn:air:...`: CivitaiのAIR
    - `org/repo@...`: HuggingFaceのリポジトリ
    - `https://...`: URL
    - `file://...`: ローカルのパス
- `quantization`: 量子化の指定
  - `none`: 量子化しない
  - `int8wo`: INT8量子化
  - `int4wo`: INT4量子化
  - `fp8wo`: FP8量子化
- `vae_tiling`: VAEのタイリングの指定
  - `true`: 有効
  - `false`: 無効
- `training_resolution`: 学習時の解像度、フロントエンドが比率計算に使う。
  - `1024`: 1024x1024
  - `1536`: 1536x1536
  - `1024x1024`: このように幅高さを指定することも可能。
- `default_steps`: 生成時のステップ数
  - 10から50程度が推奨される。
- `default_cfg`: 生成時のCFGスコア
  - 7.0程度が推奨される。
- `default_sampler`: 生成時のサンプラー
  - `euler`: Euler
  - `euler_a`: Euler Ancestral
  - `dpm++_2m`: DPM++ 2M
- `output_format`: 出力画像の形式
  - `webp`: WebP(PNGよりもファイルサイズが1/10程度になる)
  - `png`: PNG
- `prompt_parser`: Prompt Parserの指定
  - `lpw`: LPW(A1111/WebUI互換)
  - `compel`: Compel(Diffusers推奨)
