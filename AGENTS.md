# Agents Guide

This repository contains a text-to-image generation service. If you are an AI
agent, please look for detailed documentation in the `docs/` directory.

- **`docs/API_GUIDE.md`**: API endpoints, usage examples, and SSE protocol
  details.
- **`docs/MODELS.md`**: Supported model types, their fixed parameters, and VRAM
  profiles.
- **`docs/DEVELOPMENT.md`**: Instructions for development, testing, and
  deployment (Taskfile).

The `README.md` is intended for human users (Quick Start). Please prefer reading
`docs/*.md` files for deeper understanding of the system.
